# Legal Notice and Privacy Policy 2.2.6.7

This release adds a new section to the generated privacy policy about exporting data from webtrees.

Data that is exported, printed, saved, copied, or captured in a screenshot is no longer protected by the technical access controls of webtrees. The new text explains this risk and describes common ways in which data can leave webtrees:

* GEDCOM exports, including the available visibility levels such as exports without privacy filtering or exports as visible to managers, members, or visitors.
* Reports, which can export selected data as text or generated output according to the user's webtrees permissions.
* Screenshots, which are still based on the currently visible page but are no longer controlled by webtrees after they have been created.

The German translation has been updated accordingly.
